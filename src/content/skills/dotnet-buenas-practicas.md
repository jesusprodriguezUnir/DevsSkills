Aplica las mejores prácticas de la industria para el siguiente bloque de código o arquitectura:

- **Minimal APIs** vs Controllers.
- **Entity Framework Core**: Optimización de consultas, `AsNoTracking` y Global Query Filters.
- **Logging**: Implementación de `ILogger` y logging estructurado.
- **Middleware**: Configuración correcta del pipeline de solicitud.

Refactoriza el código siguiendo el principio de **Simplicity First**.
