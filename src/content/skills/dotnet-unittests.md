Genera pruebas unitarias para el código .NET proporcionado siguiendo estos lineamientos:

1. **Framework**: xUnit.
2. **Mocks**: Usa Moq para las dependencias de interfaces.
3. **Estructura**: Arrange, Act, Assert.
4. **Casos**: Incluye "Happy Path", valores nulos, excepciones y casos de borde.

Asegúrate de que las pruebas sean **Goal-Driven** y verifiquen el comportamiento esperado con precisión.
